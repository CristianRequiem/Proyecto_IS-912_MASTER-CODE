LENGUAJES

Podrás desarrollar tus proyectos bajos los siguientes lenguajes


HTML5

Es la última versión de HTML. Es el lenguaje usado para estructurar y presentar el contenido de las páginas web.

JavaScript

Es el lenguaje de programación utilizado principalmente para crear efectos atractivos y dinámicos para nuestras páginas web.

CSS3

Última versión de CSS. Es el lenguaje para definir el estilo o apariencia de las páginas web, escritas con HTML.
